# beautiful-trimming

クリップボードの切り抜き画像の上下左右の余白を均等に整える常駐ツール。

## セットアップ

```
pip install -r requirements.txt
```

## 使い方

```
python trim.py
```

起動すると常駐し、クリップボードを監視する。スニップ（例: `Win+Shift+S`）などで新しい画像がクリップボードに入ると、自動で余白を均等化して書き戻す。あとはそのまま貼り付けるだけ。ホットキー操作は不要。

`Ctrl+C` で終了する。設定（`ratio` / `poll_interval` / `tolerance` / `corner_size`）は `config.toml` で変更する。

自分が書き戻した画像はシグネチャとクリップボードのシーケンス番号で除外するため、再処理で図が縮み続けることはない。

## 切り分け（うまく動かないとき）

`diagnose.py` を使うと、監視に依存せず取得→背景推定→検出→整形→書き戻しの各段階を確認できる。整えたい図をクリップボードにコピーしてから次を実行する。

```
python diagnose.py
```
